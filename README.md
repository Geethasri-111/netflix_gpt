#Netflix clone

- npm init react-app netflix

- tailwind install 
    - npm install -D tailwindcss@3
    - npx tailwindcss init

- add in tailwind.config
    - content: ["./src/**/*.{js,jsx,ts,tsx}",],

- in index.css add
    - @tailwind base;
    - @tailwind components;
    - @tailwind utilities;

- npm start
